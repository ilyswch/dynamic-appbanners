
# Smart App Banner Generator for Apple’s App Store

A tiny web tool that injects a dynamic Smart App Banner into a page so visitors on iOS/iPadOS can view, open, or install an app from the store using only an App Store link — for environments where the native store app is hidden by MDM-policies

----------

## Quick start
### [Use my GitHub page](https://ilyswch.github.io/dynamic-appbanners/)
or
### Host yourself
Clone the repo or copy the `index.html` into your project.
 
## Usage   
1.  Open the page in **iOS/iPadOS Safari** (third-party browsers will not work).
    
2.  Paste a AppStore URL into the input and click **Apply Smart App Banner**.
